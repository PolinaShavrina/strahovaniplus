#  Система Добровольного Медицинского Страхования (ДМС) "СтрахованиеPlus"

Веб-приложение для управления процессом добровольного медицинского страхования, включающее личный кабинет клиента и панель администратора.

##  О проекте

Проект представляет собой полнофункциональную платформу для страховой компании. Позволяет клиентам оформлять полисы ДМС онлайн, получать направления к врачам и отслеживать статус услуг. Административная панель предоставляет инструменты для управления клиентами, договорами, направлениями и медицинскими учреждениями.

**Основной функционал:**
- Регистрация и авторизация пользователей (клиенты и администраторы)
- Оформление полисов ДМС по различным программам
- Генерация уникальных номеров договоров и полисов
- Управление застрахованными лицами по одному договору
- Выдача и отслеживание направлений к врачу (талоны)
- Административная панель для полного управления системой
- Генерация отчетов и статистика

## Технологии

- **Backend:** PHP (нативный, без фреймворков)
- **Frontend:** HTML5, CSS3, JavaScript (ванильный)
- **База данных:** MySQL (PDO для подключения)
- **Архитектура:** Монолитная, MVC-подобная структура
- **Сессии:** PHP Native Sessions
